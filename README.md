📌 Project Overview

This project is part of a design assessment for AI Ready School, a platform that provides a complete AI ecosystem for schools.

The goal was to audit, redesign, and build a better homepage experience within a limited time (4–5 hours), focusing on clarity, usability, and better communication of value for school decision-makers.

🧠 Step 1 — Audit (Problems Identified)

After reviewing the current website, these were the main issues:

Navigation feels crowded
Too many items in the navbar make it harder for users to quickly understand where to go.
Hero section is unclear
The main message does not strongly communicate what the product actually offers or why it matters.
Weak call-to-action (CTA)
The primary action is not visually strong or compelling enough to drive user interaction.
Content is too text-heavy
Large blocks of text reduce readability and make it harder to scan quickly.
🎨 Step 2 — Design (Figma)

I redesigned the homepage in Figma with a focus on simplicity and clarity.

Key Design Decisions
Simplified Navigation
Reduced clutter
Kept only essential links
Added a clear “Book Demo” button for quick action
Improved Hero Section
Strong, clear headline: “Bring AI into Your School — Safely & Effectively”
Supporting text explains the value in simple terms
Two clear buttons:
Primary: Book Demo
Secondary: Explore Products
Better Visual Hierarchy
Important text is larger and bolder
Supporting content is lighter and easier to scan
Consistent spacing throughout
Product Section Redesign
Each product is shown in a card layout
Short descriptions instead of long paragraphs
Easy to scan and understand quickly
Clean and Professional Look
Soft colors suitable for an education platform
Balanced spacing and alignment
Modern and approachable design
💻 Step 3 — Build (HTML & CSS)

After designing, I converted the design into a working webpage using HTML and CSS.

What I Built
Responsive navigation bar
Fully styled hero section
Product cards section
Buttons and layout matching the Figma design
Key Points
The design and code closely match each other
Clean and structured HTML
CSS used for layout, spacing, and styling
Works on both desktop and smaller screens
🎯 What I Focused On

Due to time constraints, I prioritized:

Navigation
Hero section
Product overview section

These are the most important parts for first-time users.
